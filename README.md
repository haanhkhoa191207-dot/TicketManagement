# 🚆 Train Ticket Management System

A robust command-line application built in C++ to efficiently manage train ticket sales, track customer queues, and generate station analytics. 

## 🌟 Key Features

The system operates through an interactive menu offering the following functionalities:
* **Customer Management:** Add new customers to the queue, remove them when necessary, and display the current waiting list.
* **Data Persistence:** Seamlessly load (`Load customers from file`) and save (`Save customers to file`) records to ensure no data is lost between sessions.
* **Ticketing Operations:** Process and execute ticket sales.
* **Station Analytics:** 
  * Monitor real-time waiting stations.
  * Display specific stations alongside their current customer count.
  * Generate comprehensive sales statistics and revenue tracking.

## 🛠️ Tech Stack & Data Structures

* **Language:** C++
* **Core Concepts:** 
  * **Doubly Linked List:** Utilized for dynamic memory allocation and efficient customer queue management (allowing quick traversal and removal of nodes).
  * File I/O for data storage.
